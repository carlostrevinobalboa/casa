# Roadmap

## Fase 1 - Base tecnica (hecha)
- Monorepo backend/frontend
- Docker Compose con PostgreSQL y Redis
- Seguridad base y modelo multiusuario-hogar compartido
- Dashboard inicial con rutas por modulo

## Fase 2 - MVP funcional
- Despensa: CRUD, stock minimo, caducidades, alertas
- Recetas: CRUD, ingredientes, ejecucion y descuento automatico (hecho)
- Compra: lista inteligente + marcaje de compra (hecho)
- Compra: historico de precios + registro de precio real (hecho)
- Notificaciones en tiempo real (WebSocket) por usuario (hecho)

## Fase 3 - Mascotas y actividad
- Mascotas: datos base, comida, cuidados y recordatorios (en progreso, MVP backend/frontend hecho)
- Paseos con GPS + mapa + estadisticas semanales
- Actividades deportivas con GPS + historico + estadisticas

## Fase 4 - Calendario inteligente
- Vista mensual/semanal/diaria
- Eventos recurrentes
- Colores por categoria
- Recordatorios por usuario

## Fase 5 - Movil y automatizaciones avanzadas
- PWA completa: instalacion, camara, GPS, notificaciones push
- Subida ticket y OCR (si se integra)
- Automatizaciones cruzadas entre modulos
